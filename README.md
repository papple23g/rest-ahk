## Development
For convenience a `.nvmrc` file has been provided at the root of the project directory.

### Project setup
```bash
npm install
```

#### Compiles and hot-reloads for development
```bash
npm run dev
```

### Compiles and minifies for production
```bash
npm run build
```

### Compiles and runs for production
```bash
npm run start
```

### Run your tests
```bash
npm run test
```

### Lints files
```bash
npm run lint
```
